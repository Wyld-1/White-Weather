# NOAA Weather
Brings accuracy of NOAA weather to a familiar, easy-to-read application format.

### Features
* **NOAA forecasts:** Displays the same information you see on NOAA's webiste.
* **Ski resorts in search:** Instant search and weather tracking for mountain peaks, not just cities. 
* **Snow accumaltion, front and center:** Shows snow accumulation on the front page, so you don't have to hunt for the powder.
* **Dynamic backgrounds:** Information displayed over stunning slow-motion videos to instantly get the weather at a glance.
